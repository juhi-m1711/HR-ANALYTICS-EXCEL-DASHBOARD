## Project Overview  
This project involves building an **HR Analytics Dashboard using Microsoft Excel** to analyze employee data and understand workforce trends.  
It provides insights into **employee attrition, demographics, and job satisfaction** through interactive visuals and filters.  
The project was developed as a **student practice project** to strengthen Excel-based data analysis and dashboard skills.

---

## Problem Statement  
Organizations often face difficulty in identifying the key factors behind employee attrition due to unstructured HR data.  
This project aims to analyze HR data and visualize **attrition drivers across departments, job roles, and demographics** to support better HR decision-making.

---

## Objectives  
- Analyze employee attrition and workforce metrics  
- Identify high-attrition departments and job roles  
- Study attrition patterns across demographics  
- Present insights using an interactive dashboard  

---

## Key Insights  
- Total Employees, Active Employees & Attrition Rate  
- Average Age & Job Satisfaction  
- Attrition by Gender, Department, Education, Job Role, Age Group, and Marital Status  

---

## Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Charts & Visualizations  
- Slicers / Filters
